# Discord Channel Logger
Uses a discord bot to log everything in a specified channel

This application requires a little bit of set up to become functional.

Download the program and make sure it is in a folder because it creates files.

When the application opens, when you hit start for the first time, the program will create a config file and open it.

You will need two things:

Bot Token

Channel ID

-------------------------|BOT CREATION AND TOKEN|------------------------------

Go to https://discord.com/developers/applications

Hit New Application and name your bot.
(Example: Discord Channel Logger)

Go to installation

Copy the following:

User Install: Unchecked

Guild Install: Checked

Install Link: None

Go to Bot

Hit Reset Token and copy the new token (KEEP THIS PRIVATE, ANYONE CAN USE YOUR BOT WITH IT)

paste this token in the config that is created after you hit start for the first time.

STILL IN THE BOT SECTION

Copy the following:

Public Bot: Unchecked

Presence Intent: Checked

Server Members Intent: Checked

Message Content Intent: Checked

Copy  this link https://discord.com/oauth2/authorize?client_id=

Paste in url

Go to General Informaion

Under Application ID hit copy

Paste in url after = and hit enter

This will redirect you to discord and then you can add the bot to your server

-----------------------------|CHANNEL ID|-----------------------------------

For this step you will have to go into developer mode:

To enable Developer Mode in Discord,

open User Settings (the gear icon), navigate to the Advanced tab under "App Settings," and toggle on Developer Mode

go to the channel you would like to grab the id from and right click it, at the bottom it should say "Copy Channel ID"

paste that into the config in place of the placeholder numbers

hit start and the bot should activate!
